lets create layout using bootstrap. whie creating layouts grid system is something you should know about. Grid system consist of container, these containers will have rows within them and these rows will have columns within them. so every grid in bootstrap is actually a 12 column grid. so if you want to place an element in a row then you can specify its positon and its width in terms of columns. suppose I want it to occupy 25% of the area then out of 12 25% means 3 columns.
